Customer Retention & Churn Analysis Project

Project Overview

This project analyzes customer retention and churn patterns using subscription customer data. The analysis helps identify churn trends, retention drivers, and customer behavior insights.

- Churn_Pie_Chart.png
- Churn_Bar_Chart.png
- Customer_Retention_Report.pdf

Tools Used

- Microsoft Excel
- Data Analysis
- Data Visualization
- Dashboard & Reporting

Key Insights

- Customer churn rate analysis
- Subscription plan-wise churn comparison
- Regional customer distribution
- Customer retention trends
- Actionable recommendations for reducing churn

Conclusion

The analysis highlights customer retention patterns and identifies areas where churn can be reduced through improved engagement and customer support strategies.
